2-2. 共通して取得する情報（広告 SDK 経由）

本アプリのうち広告を表示するものでは、Google AdMob SDK が以下の情報を取得・送信します。

  • 広告 ID（Android Advertising ID / AAID）
  • IP アドレス
  • デバイス情報（機種・OS バージョン・言語・タイムゾーン）
  • おおよその位置情報（IP アドレスに基づく国レベル）
  • アプリの操作・広告閲覧ログ

3. 利用目的

当方は、個人情報保護法第 17 条に基づき、取得した情報を以下の目的の範囲内でのみ利用します。

  • 本アプリの機能提供（利用者が入力したデータの記録・表示・集計）
  • 広告の配信および最適化（Google AdMob 経由）
  • アプリの不具合解析・品質改善（Firebase Crashlytics 経由）
  • 有料機能の提供（ADHDセルフチェック）
  • お問い合わせへの対応
  • 本ポリシーの変更通知等の連絡

取得した情報を上記目的以外に利用する場合は、事前に利用者の同意を取得します。

4. 第三者提供（送信先 SDK 一覧）

本アプリでは、以下の第三者サービス（SDK）を利用しており、各サービスへ利用者情報が送信されます。いずれの場合も、当方が利用者の氏名・住所・電話番号・メールアドレス等を直接送信することはありません。

4-1. Google AdMob

  • 提供事業者: Google LLC
  • 送信先国: アメリカ合衆国
  • 送信される情報: 広告 ID（AAID）、IP アドレス、デバイス情報（機種・OS バージョン）、おおよその位置情報（国レベル）、アプリ操作・広告閲覧ログ
  • 送信目的: 広告配信、不正クリック検知、効果測定
  • プライバシーポリシー: https://policies.google.com/privacy
  • 広告ポリシー: https://policies.google.com/technologies/ads

4-2. Firebase Crashlytics

  • 提供事業者: Google LLC
  • 送信先国: アメリカ合衆国
  • 送信される情報: クラッシュ時のスタックトレース、デバイス情報、アプリ識別子、Firebase インストール ID、アプリ状態
  • 送信目的: アプリの不具合解析・品質改善
  • プライバシーポリシー: https://firebase.google.com/support/privacy

4-3. Google SpeechRecognizer（まなび電卓のみ）

  • 提供事業者: Google LLC
  • 送信先国: アメリカ合衆国
  • 送信される情報: 音声入力データ（数式読み上げ）
  • 送信目的: Android 標準の音声認識機能による文字列変換
  • 備考: 当方は音声データを取得・保存しません。Android OS が SpeechRecognizer API 経由で Google に送信します。

4-4. Firebase Authentication / Cloud Firestore（うちのこログのみ）

  • 提供事業者: Google LLC
  • 送信先国: アメリカ合衆国
  • 送信される情報: Google アカウント UID、家族グループに共有するペット情報・写真
  • 送信目的: 家族間でのデータ共有機能の提供

4-5. Google Play Billing（ADHDセルフチェックのみ）

  • 提供事業者: Google LLC
  • 送信される情報: 課金トークン、購入状態（premium_unlock）
  • 送信目的: 有料機能（¥500 一括課金）の提供と購入状態の検証

5. 広告について

本アプリの一部では、Google AdMob を利用して広告を表示しています。広告配信に伴うデータ取扱いは、第 4-1 項に記載のとおりです。

5-1. EU/EEA・英国における同意取得（UMP）

EU/EEA・英国の利用者に対しては、Google User Messaging Platform（UMP）を用いて、広告に関するデータ利用への同意を初回起動時に取得します。利用者は「パーソナライズド広告」または「非パーソナライズド広告」のいずれかを選択でき、設定はアプリ内の「同意設定をやり直す」メニューからいつでも変更できます。

5-2. 広告 ID のリセット

利用者は Android 端末の「設定 → プライバシー → 広告」から、広告 ID をいつでもリセット・削除できます。

6. 要配慮個人情報の取扱

個人情報保護法第 2 条第 3 項に定める「要配慮個人情報」（病歴、心身の機能の障害、医師等による健康診断等の結果等）に該当しうる情報を取り扱う本アプリについて、当方は以下のとおり厳格に管理します。

  • のんだっけ?: 服薬情報は端末内の DataStore にのみ保存され、当方および第三者へ送信されません。
  • 健康ログ: 体重・血圧・脈拍・体脂肪率は端末内の Room データベースにのみ保存され、当方および第三者へ送信されません。
  • ADHDセルフチェック: ASRS / WURS / AQ-10 の心理検査スコアは端末内の SharedPreferences にのみ保存され、当方および第三者へ送信されません。
  • エゴグラム診断: 性格診断履歴は端末内の Room データベースにのみ保存され、当方および第三者へ送信されません。

これらの要配慮個人情報は、利用者の端末内に閉じた状態で保管され、利用者がアプリをアンインストールすることで完全に削除されます。当方が広告配信・クラッシュ解析の目的でこれらの情報を取得・送信することはありません。

7. お子様の個人情報（児童保護）

当方は、Google Play の「ファミリー向けプログラム ポリシー」および COPPA（米国児童オンラインプライバシー保護法）の趣旨に従い、児童の個人情報保護に努めます。

  • あいうえおかきかた（Kakikata）: 4〜7 歳の児童を対象とした完全オフラインアプリです。INTERNET 権限を保持せず、いかなる情報も収集・送信しません。広告も表示しません。
  • はじめてごはん（HajimeteGohan）: 離乳食を扱う乳児の保護者向けアプリであり、児童本人による直接利用を想定していません。データは端末内に保存されます。
  • おてつだいBank、おくだけ勇者等: 児童が利用する可能性がある場合でも、児童から個人を特定する情報を意図的に収集することはありません。

13 歳未満の児童が誤って個人情報を提供したことが判明した場合、当方は速やかに削除いたします。contact@akiappsdev.com までご連絡ください。

8. データの保存・保持期間

  • 端末内データ: 利用者が本アプリをアンインストールした時点で完全に削除されます。当方はこれらのデータをバックアップ・回復する手段を有しません。
  • Google AdMob / Firebase Crashlytics 経由のデータ: Google LLC のデータ保持ポリシーに従って保管されます。詳細は Google プライバシーポリシーをご参照ください。

11. 安全管理措置

当方は、取得した利用者情報の漏えい・滅失・毀損の防止のため、以下の安全管理措置を講じています。

  • 技術的安全管理措置: 利用者が入力したデータは原則として端末内に保存し、外部送信を行いません。外部通信が必要な場合は HTTPS（TLS）による暗号化通信のみを使用します。Firebase 等のクラウドサービスは、Google が提供する暗号化保管・アクセス制御を利用します。

  • 物理的安全管理措置: 開発端末はパスワード／生体認証によるロック、ディスク暗号化を実施しています。
  • 人的安全管理措置: 事業主本人が個人情報保護法および GDPR に関する自主的な学習を継続しています。
